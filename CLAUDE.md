# ことだまリドル - 開発メモ

## プロジェクト概要
- ひらがな単語当てゲーム（Wordle風）
- 単一ファイル構成: `index.html`（HTML/CSS/JS をすべて含む）

## コード規約（必須）
- HTML/CSS/JS 内は役割ごとにセクション分割し、統一フォーマットの区切りコメントを付ける
  例: `/* ===== SECTION: セクション名 ===== */`
- 既存セクションを編集する際は、ファイル全体を読み込まず、該当の区切りコメントを Grep で検索してから必要範囲だけを読み書きする
- 新規コードも必ずいずれかのセクションに分類して書く。迷う場合は最も近いセクションに含め、一言コメントで補足する

## 作業時の振る舞い（トークン節約）
- 変更内容を説明する際は、変更箇所と該当セクション名だけを示し、関係のないコード全文は貼り付けない
- 大きな改修に着手する前は、簡潔な方針を一言提示してから進める

## やってはいけないこと
- 純粋なリファクタリング（コメント追加・整形）以外で、既存ロジックの挙動を事前確認なしに変更しない
